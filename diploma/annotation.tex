\section*{Аннотация}
Данная работа описывает реализацию протоколов в языке \lang{Kotlin} для платформы \tool{JVM}. Для реализации используется библиотека вызовов из \lang{Java}, механизм разрешения перегрузок и кэширование. Представлены измерения производительности в различных случаях. Улучшено время вызова в случаях многих аргументов. Расширены границы применимости, рассмотрена возможность реализации полноценной структурной типизации.
